== Description ==

Seo Keyword Page is a Free Wordpress plugin which can automatically link keywords or phrases in posts, pages, categories, comments and tags on your site.
Seo Keyword Page provides an automatic SEO solution for your wordpress site in addition to custom keyword lists, no-follow and much more.

Seo Keyword Page allows you to set up your own preferred keywords and set of matching URL's. This plugin is perfect solution to get your posts and pages interlinked or simply add affiliate links automatically.

To edit the settings of the plugin visit -> WP Admin -> Seo Keyword Page

== Installation ==

1. Upload seo-keyword-page folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. Use the Options page to modify default options if required.
